# WSAUpdateChecker
A tool which can generate download links for WSA in MS store and store the package information.  
Only support Windows and python3.

# Usage:
1. Make sure you have installed Python3 and install all necessary library in the computer. The development environment is Python 3.11.3 and in Windows 11.
2. Check all necessary library and install which is miss.
3. Run "generateWSALinks.bat" to start up this tool with a console
4. You also can run "WSAUpdateChecker.vbs" to start up silently
